# WinUI 3 File Copier

A modern, fast file utility built with C# and WinUI 3.

## Features
- **Modern UI:** Built using Fluent Design principles.
- **Asynchronous I/O:** Won't freeze your computer during large transfers.
- **State Management:** Fully supports pausing and canceling mid-operation.

## How to Use
1. Launch the app.
2. Drag files or folders into the gray dashed area.
3. Click "Select Destination" to choose where files should go.
4. (Optional) Check "Delete source files" to move instead of copy.
5. Click **Start**.
