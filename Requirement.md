Project Name: WinUI 3 Modern File Copier

Target Framework: .NET 8 / Windows App SDK 1.5+

Core Functionalities:

- Drag & Drop: Support for multiple files and folders into a designated drop zone.

- Folder Selection: Integration with Windows FolderPicker for destination path.

- Operation Modes: Choice between "Copy" and "Move" (Delete source after copy).

- Execution Control: Real-time Start, Pause, Resume, and Cancel functionality.

- Progress Feedback: Visual progress bar and status text updates.
